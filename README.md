# Sản phẩm biến thể

## 1. Mục đích

Dùng để sử dụng cho các ngành hàng có sản phẩm với các thuộc tính khác nhau như màu sắc, kích thước, chất liệu…

# 2. Hướng dẫn sử dụng web
